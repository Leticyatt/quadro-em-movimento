# Quadro em Movimento

Um sistema de gerenciamento de tarefas (Kanban) moderno, inspirado na interface e usabilidade do **Trello**, focado em oferecer uma excelente experiência do usuário (UX) e agilidade no fluxo de trabalho.

**Acesse o projeto online aqui >> (https://Leticyatt.github.io/quadro-em-movimento/)**

### Funcionalidades:
* **Drag & Drop:** Organize suas tarefas movendo-as entre as listas.
* **Edição Inline:** Clique no texto de qualquer cartão para editá-lo instantaneamente.
* **Modo Escuro:** Conforto visual com um clique no toggle.
* **Persistência Local:** Suas tarefas e listas ficam salvas no navegador (LocalStorage).
* **Foco em Atalhos:** Exclua listas rapidamente usando a tecla Backspace".
* **Segurança de Ação:** Sistema de "Desfazer" (Undo) ao excluir tarefas.

### Tecnologias
* HTML5 / CSS3 (Variáveis CSS e Flexbox)
* JavaScript Vanilla (DOM nativo, Delegação de Eventos)
